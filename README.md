## 屏蔽文学类内容农场

### 什么样的网站会出现在这里？
 - 有百度谷歌等的广告
 - 无备案信息，或查询不到
 - 使用同一个模板网站
 - 大量网站，混淆视听
 - 题文不符
 - 网站内容/版式过旧
 - 出现在谷歌搜索前几页
